================================================================
   🌿 心灵驿站 — 在线解压网站 · 项目说明
================================================================

【如何打开】
  双击 index.html 即可在浏览器中打开网站。
  推荐使用 Chrome、Edge 或 Firefox，体验更佳。

================================================================
  项目结构
================================================================

  d:\开发网站\
  ├── index.html          ← 📄 主页面（双击打开）
  ├── 样式\
  │   └── 主样式.css       ← 🎨 外观样式
  ├── 脚本\
  │   └── 主脚本.js        ← ⚡ 交互逻辑
  ├── 图片\               ← 🖼️ 图片资源（可自行添加）
  └── 项目说明.txt         ← 📖 本说明文件

================================================================
  网站功能一览
================================================================

【🧘 心理舒缓小贴士】（心理咨询区域）
  包含 6 个实用的减压方法：
    1. 478 呼吸法     — 科学调节心率
    2. 情绪日记       — 把情绪「倒」到纸上
    3. 正念散步       — 回到当下的练习
    4. 音乐疗愈       — 用声音放松大脑
    5. 自我拥抱       — 刺激催产素分泌
    6. 温暖饮品       — 触觉传递暖意

【🎮 解压小游戏】（4 个互动游戏）
  1. 🫧 捏泡泡  — 点破屏幕上的泡泡，听到「啵」声超解压
  2. 🎨 自由涂鸦 — 在画布上随意画画，释放情绪
  3. 🌬️ 呼吸引导 — 跟着动画一起深呼吸（4-7-8 呼吸法）
  4. 🔢 疯狂点击 — 10 秒挑战，看你能点多少次

【🎵 放松音效】
  4 种可切换的背景音效：
    - 雨声（助眠安神）
    - 海浪声（平静心灵）
    - 森林鸟鸣（置身自然）
    - 轻柔钢琴（舒缓旋律）
  支持同时只播放一个，再次点击暂停。

【✨ 每日一句】
  12 条精选正能量语录，点击按钮随机切换。

【📱 移动端适配】
  在手机和平板上也能流畅使用，布局自动调整。

================================================================
  技术说明
================================================================

  技术栈：HTML5 + CSS3 + 原生 JavaScript
  无需安装任何依赖，无需联网（除在线音频外）。
  音频来源：Pixabay 免费音效库。

  如需离线使用音频：
    1. 下载 mp3 文件放入「音频」文件夹
    2. 修改 index.html 中 <audio> 的 src 路径

================================================================
  自定义修改指南
================================================================

  改颜色   → 编辑 主样式.css 顶部的 :root 变量
  改文字   → 编辑 index.html 中的文本内容
  改名言   → 编辑 主脚本.js 底部的「名言列表」数组
  改游戏   → 编辑 主脚本.js 中对应的游戏函数
  加功能   → 在 index.html 添加 HTML，在对应 CSS/JS 文件添加代码
